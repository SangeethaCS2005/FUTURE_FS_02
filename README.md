# Client Lead Management System (Mini CRM)

## 📌 Project Overview

The Client Lead Management System (Mini CRM) is a simple web-based application developed as part of the **Future Interns Full Stack Development Internship**.

The application allows users to manage client leads by adding, viewing, updating, and deleting lead information. The frontend is developed using HTML, CSS, and JavaScript, while the backend was developed using Node.js, Express.js, and MongoDB.

> **Note:** The frontend is deployed using GitHub Pages. The backend is included in this repository for reference but is not deployed.

---

## 🚀 Features

* Add new client leads
* View all leads
* Update existing leads
* Delete leads
* Responsive user interface
* Simple and clean design

## Backend Setup (Optional)

If you want to run the backend locally:

```bash
cd backend
npm install
node server.js
```

Configure your own MongoDB Atlas connection string before running the backend.
